数据类型

1.1) ECMAScript中有5种简单数据类型（也称为基本数据类型）

Undefined,Null,Boolean,Number,string。还有一种复杂数据类型Object.

1.2) typoe操作符

鉴于ECMAScript是松散型的，因此需要有一种手段来检测给定变量的数据类型---typeof就是负责提供这方面信息的操作符。

undefined------未定义
boolean------布尔值
string-------字符串
number--------数值
object--------对象或null
function-------函数
typeof例子：
var message = "some string";
alert (typeof message); //"string"
alert (typeof (message)); //"string"
alert (typeof 95); //"number"
alert (typeof null) //object 空对象;

1.3) Undefined类型

在使用var声明变量但未对其加以初始化时，这个变量的值就是undefined
var message;
alert (message == undefined); //true

例子：
var message; //这个变量声明
var age //这个变量没有声明

alert(message);//"undefined";
alert(age);//产生错误

alert(typeof message);//"undefined"
alert(typeof age);//"undefined"

1.4) Null类型

var car = null;设置为null将来要保存对象
alert(typeof car);//object空对象
实际上undefined是派生字null值的，因此、ECMAScript-262相等性测试：

Boolean是ECMAScript用的最多的一种类型，该类型只有两个字面值：true&&false,true不一定等于1，false不一定等于0

1.6) Number 类型

不要做这样的测试：
0.1+0.2的结果不是0.3，而是0.30000000004。
if(0.1+0.2 == 0.3){
alert("you got 0.3");
}
数值转换；
var num1 = Number("Hello world"); //NaN
var num2 = Number(""); //0
\ var num3 = Number("00001"); //1
var num4 = Number(true); //1
var num5 = Number(false); //0
由于Number()函数在转换字符串时比较复杂而且不够合理,所有一半处理整数用parseInt

字符串可以由双引号""和单引号表示''表示
